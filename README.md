# Şenoğlu Hukuk Bürosu — Next.js

İstanbul merkezli Şenoğlu Hukuk Bürosu için Next.js 14 (App Router) + TypeScript ile geliştirilmiş kurumsal web sitesi.

![Ekran görüntüsü](screenshot.png)

## Teknolojiler

- Next.js 14 (App Router)
- React 18
- TypeScript

## Geliştirme

```bash
npm install
npm run dev
```

Uygulama `http://localhost:3000` adresinde çalışır.

## Komutlar

- `npm run dev` — geliştirme sunucusu
- `npm run build` — üretim derlemesi
- `npm start` — üretim sunucusu
